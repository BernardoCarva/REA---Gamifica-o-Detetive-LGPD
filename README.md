# REA---Gamifica-o-Detetive-LGPD
Recurso Educacional Aberto “Detetive Digital – O Caso dos Dados Vazados”, uma aula gamificada sobre a LGPD (Lei nº 13.709/2018). Promove o aprendizado prático e reflexivo sobre privacidade, segurança da informação e responsabilidade digital, integrando aspectos técnicos e educacionais.
